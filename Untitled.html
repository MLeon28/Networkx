Creando un grafo¶

Por definicin, un grfico es una coleccin de nodos (vrtices) junto con pares de nodos identificados (llamados aristas, enlaces, etc.). En NetworkX, los nodos pueden ser cualquier objeto modificable, por ejemplo, una cadena de texto, una imagen, un objeto XML, otro grfico, un objeto de nodo personalizado, etc.

In [2]:
import networkx as nx
import matplotlib.pyplot as plt
G = nx.Graph()

Nodos

La grfica G puede crecer de varias maneras. NetworkX incluye muchas funciones de generacin de grficos y facilidades para leer y escribir grficos en muchos formatos. Sin embargo, para comenzar, veremos manipulaciones simples. Puede agregar un nodo a la vez,

In [3]:
G.add_node(1)

o agregue nodos desde cualquier contenedor iterable, como una lista

In [4]:
G.add_nodes_from([2, 3])

Tambin puede agregar nodos junto con atributos de nodos si su contenedor produce 2 tuplas de la forma (nodo, nodo_atributo_dict):

In [5]:
G.add_nodes_from([

    (4, {"color": "red"}),

    (5, {"color": "green"}),

])

Los atributos de los nodos se analizan ms adelante.

Los nodos de un grfico se pueden incorporar a otro:

In [6]:
H = nx.path_graph(10)

G.add_nodes_from(H)

G ahora contiene los nodos de H como nodos de G. Por el contrario, podra usar el grfico H como un nodo en G.

In [7]:
G.add_node(H)

In [8]:
G = nx.Graph()
G.add_edge(1, 2)
G.add_edge(1, 3)
G.add_edge(1, 5)
G.add_edge(2, 3)
G.add_edge(3, 4)
G.add_edge(4, 5)

# explicitly set positions
pos = {1: (0, 0), 2: (-1, 0.3), 3: (2, 0.17), 4: (4, 0.255), 5: (5, 0.03)}

options = {
    "font_size": 36,
    "node_size": 3000,
    "node_color": "white",
    "edgecolors": "black",
    "linewidths": 5,
    "width": 5,
}
nx.draw_networkx(G, pos, **options)

# Set margins for the axes so that nodes aren't clipped
ax = plt.gca()
ax.margins(0.20)
plt.axis("off")
plt.show()

Creando un grafo dirigido

In [10]:
G = nx.DiGraph([(0, 3), (1, 3), (2, 4), (3, 5), (3, 6), (4, 6), (5, 6)])

# group nodes by column
left_nodes = [0, 1, 2]
middle_nodes = [3, 4]
right_nodes = [5, 6]

# set the position according to column (x-coord)
pos = {n: (0, i) for i, n in enumerate(left_nodes)}
pos.update({n: (1, i + 0.5) for i, n in enumerate(middle_nodes)})
pos.update({n: (2, i + 0.5) for i, n in enumerate(right_nodes)})

nx.draw_networkx(G, pos, **options)

# Set margins for the axes so that nodes aren't clipped
ax = plt.gca()
ax.margins(0.20)
plt.axis("off")
plt.show()
